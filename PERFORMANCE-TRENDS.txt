KEY TAKEAWAYS

1. SMOLLM3 IS THE CLEAR WINNER
   - Dominates every test (6/6 first place)
   - 2x faster than 8B models
   - Stable across all input sizes
   - Only 4% degradation from 10 → 2000+ tokens

2. 8B MODELS ARE THE SWEET SPOT
   - llama-3.1-8b and hermes-3-llama-8b ultra-stable
   - Better quality output than smollm3 (expected from larger size)
   - Still 54-55 tok/sec (6x faster than 24B+ models)
   - ±1% variance means predictable performance

3. BIGGER IS NOT BETTER
   - 24B models: 16-18 tok/sec (6x slower than smollm3)
   - 32B models: 16-20 tok/sec (7x slower than smollm3)
   - Worst scaling: qwen2.5-32b (-21.7% in Run 6)
   - Resource cost not justified by performance

4. SCALING CONFIRMS PRODUCTION REALITY
   - Run 6 multi-tier (2000+ tokens) realistic production test
   - All models maintain performance (except large ones degrade)
   - Measurements from Run 1-5 predict Run 6 accurately
   - Performance metrics are stable and reliable

5. DEPLOYMENT RECOMMENDATION
   PRIMARY:    smollm3 (110+ tok/sec, lowest resource cost)
   SECONDARY:  llama-3.1-8b (55 tok/sec, best stability)
   TERTIARY:   hermes-3-llama-8b (54 tok/sec, most stable)
   AVOID:      All 24B+ models (diminishing returns)
